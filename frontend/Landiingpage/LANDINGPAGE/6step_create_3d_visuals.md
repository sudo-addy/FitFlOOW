# STEP 6 — CREATE 3D VISUALS

Create immersive visuals around the uploaded subject.

Never replace the uploaded image.

Possible enhancements:
- Floating lighting
- Ambient particles
- Glass reflections
- Depth
- Scroll animations
- Mouse parallax
- Perspective movement
- Soft shadows
- Bloom
- Gradient lighting

Use React Three Fiber when appropriate.

Avoid over-animation.
Everything must remain smooth.
60 FPS target.
